# Year Progress Wallpaper

A minimal, full-screen visualization of the year's progress, designed to be used as a dynamic wallpaper.

## Features
- **365-Dot Grid**: Visualizes every day of the year.
- **Dynamic Updates**: Highlights the current day and fills past days.
- **Theming**: Integrated support for **Dark**, **Light**, and **Cyberpunk** themes.
- **Responsive**: Perfectly scales to desktop and mobile screens.

## How to Deploy (GitHub Pages)
1.  Fork this repository or upload the files to a new GitHub repository.
2.  Go to **Settings** > **Pages**.
3.  Select the `main` branch as the source and save.
4.  Your site will be live at `https://<your-username>.github.io/<repo-name>/`.

## How to Use as Wallpaper
Once deployed, you can use the URL to set it as a live wallpaper on your devices.

### Android
- Use **Wallpaper Engine** (Recommended) or **Lively Wallpaper**.
- Select "Open URL" and paste your deployment link.
- Select your preferred theme on the screen; it will be remembered!

### iOS
- Use **Shortcuts** to fetch the page image daily, or use a specialized "HTML Wallpaper" app if available.

### Desktop (Windows)
- Use **Wallpaper Engine**.
- Select "Open from URL".
